¿Qué es Regor?

Es un sistema de riego inteligente impulsado por energía solar, diseñado para mejorar la eficiencia del riego y reducir los costos de energía. Con el uso de sensores de humedad, regula la cantidad de agua suministrada a las plantas, mientras que los paneles solares generan la energía necesaria. A través de una página web o una aplicación móvil, los usuarios pueden monitorear y ajustar el sistema en tiempo real, facilitando un control remoto eficiente. Además, proporciona la capacidad de revisar el historial de riego, recibir alertas y acceder a los servicios de manera conveniente desde dispositivos móviles.

Propósito de Solar Hydro

Es proporcionar un sistema de riego inteligente y sostenible que optimiza el uso del agua y reduce los costos de energía. Mediante el uso de energía solar y sensores de humedad, el sistema permite un control remoto eficiente del riego. Tanto la página web como la aplicación móvil permiten a los usuarios monitorear y ajustar el sistema en tiempo real, visualizar el historial de riego y recibir alertas. Además, la aplicación móvil ofrece la comodidad de acceder a estos servicios sin necesidad de una computadora.

Aplicación móvil

  • Notificaciones: Agregaremos notificaciones en la aplicación móvil para alertarte cuando el sistema de riego se active o cuando haya algún problema con el sistema.
  • Mostrar datos en tiempo real: Mostrará los datos de humedad del suelo, la cantidad de agua suministrada y otros datos relevantes en tiempo real en la aplicación móvil. Esto permitiría monitorear el sistema de riego y ajustar la configuración según sea necesario.
  • Control remoto: Es una función de control remoto en la aplicación móvil para activar o desactivar el sistema de riego desde cualquier lugar.
  • Historial de riego: Mostrará un historial de riego en la aplicación móvil para que puedas ver cuántas veces se ha activado el sistema de riego y cuánta agua se ha suministrado a las plantas.
  • Información de contacto: Agregaremos información de contacto en la aplicación móvil para que los usuarios puedan comunicarse con nosotros si tienen alguna pregunta o problema.
